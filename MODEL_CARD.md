# Model Card

## Описание модели
**Тип модели:** Logistic Regression (Логистическая регрессия)
**Версия:** emo-logreg-v1
**Фреймворк:** Scikit-learn

## Назначение
Модель предназначена для автоматической классификации текстовых сообщений (постов) из социальной сети ВКонтакте на 2 класса (авиация и автотранспорт).
*   **Вход:** Текстовая строка (str).
*   **Выход:** Метка класса (int/str) и вероятность (float).

## Обучение
### Параметры
Основная конфигурация (из `train_config.yaml`):
*   `max_iter`: 10000
*   `solver`: lbfgs
*   `C`: 1.0
*   `penalty`: None

### Процесс обучения
Обучение запускается через DVC пайплайн:
```bash
dvc exp run -n <experiment_name> --set-param exp_name=<experiment_name>
```
