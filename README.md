# Полезные команды

## Новый эксперимент:
```bash
dvc exp run -n <name>
dvc exp push origin <name>
```

## Воспроизвести эксперимент:
```bash
dvc exp apply <name>
```

## Запуск MLflow:
`mlflow server`

## Запуск сервиса:
`docker compose up`
`-d` – скрыть логи
`--build` – пересобрать образ (если были изменения в коде)
